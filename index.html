Oilfilled Job Shop

Skills: HTML/CSS, Javascript

About: Coding challenge sent by Adster Creative. I was tasked to create a website based on this PNG file that they gave me, with a deadline of 6 days (Feb. 19 - Feb. 26 2019). There was four sections in which I was evaluated: SEO, download speed, responsiveness and coding neatness. For those categories, I implemented a description meta tag for the SEO, coded only HTML/CSS and VanillaJS and a few google fonts and font awesome to limit lag on download speed. For responsiveness, I started off with a desktop design considering the PNG that was handed to me. Afterwards, it was tweaked via media queries to make it look good in other screen sizes. As for the code, I seperated it into sections following the flow of the PNG. It's also commented and properly indented if another coder wished to look at the build. This is also why the CSS wasn't minimized.
